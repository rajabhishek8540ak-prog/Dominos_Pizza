**Domino's_Pizza**

A data-driven project designed to perform end-to-end sales analysis for Domino’s Pizza using MySQL for data processing and Power BI for interactive business insights. The project focuses on uncovering performance trends, improving decision-making, and optimizing sales strategies across categories and time periods.

**Project Overview**

This dynamic analytics project involves extracting, transforming and analyzing raw Domino’s sales data to provide actionable insights. Using MySQL, the dataset is cleaned, modeled and queried to answer key business questions. The processed data is visualized in Power BI, enabling stakeholders to monitor KPIs and track sales performance in real time.
